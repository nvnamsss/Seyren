## In Scope
- Labs scene/controller updates required to visualize Ability features.
- Labs scene/controller updates required to visualize Projectile features.
- Labs UI updates required to visualize runtime state changes.
- Documentation and setup instructions for running the v0.2.0 demonstrations.

## Out of Scope
- Gameplay balance tuning.
- Production-level UI polish and final art direction.
- Broad performance optimization not required for visualization goals.
- Feature expansion unrelated to Ability, Projectile, or UI visualization.

## Acceptance Criteria
- Ability visualization works in Labs for at least one end-to-end cast flow.
- Projectile visualization works in Labs for Common, Homing, Target, and Bezier types.
- UI visualization updates correctly from Seyren runtime state changes.
- One reproducible demo path shows Ability + Projectile + UI without runtime exceptions.
- This version doc includes setup instructions and known limitations.
